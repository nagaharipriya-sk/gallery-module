## Project Overview
This is a full-stack web application developed using the MERN stack. It includes dynamic gallery management and events/announcements handling with admin features.

## Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## Features
- Dynamic Gallery Page
- Events / Announcements Page
- Image Upload Functionality
- Admin Dashboard
- CRUD Operations for Gallery & Events

## API Endpoints
### Gallery
- GET /gallery
- POST /gallery (Image Upload)
- DELETE /gallery/:id

### Events
- GET /events
- POST /events
- PUT /events/:id

## Folder Structure
/client → React frontend  
/server → Node.js backend  

## Installation
1. Clone the repository  
2. Run `npm install` in both client & server  
3. Start backend: `npm start`  
4. Start frontend: `npm start`
